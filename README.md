# invito_cilla

Invito one-page (mobile-first) — festa di laurea.

- `index.html` — pagina unica (HTML + CSS + JS inline), scroll-snap verticale
- `assets/invito.mp4` — video introduttivo (H.264 + audio)
- `assets/poster.jpg` — fotogramma poster del video
- `assets/laurel-divider.svg` — divisore ad alloro dorato

## Anteprima locale

```bash
python3 -m http.server 8000
# poi apri http://localhost:8000
```

## Deploy su GitHub Pages

1. **Settings → Pages**
2. **Source:** `Deploy from a branch`
3. **Branch:** `main` / `/ (root)` → Save

> Nota: GitHub Pages su repository **privati** richiede un piano GitHub Pro/Team.
> Con un account Free, per pubblicare le Pages il repo va reso pubblico.
